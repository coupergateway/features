# Couper Installer

This repository provides resources and instructions for installing [Couper](https://github.com/coupergateway/couper), a lightweight API gateway designed to support developers in building and operating API-driven web projects.

## About Couper

Couper is an open-source API gateway focused on simplicity and developer productivity. For more details, visit the [Couper GitHub repository](https://github.com/coupergateway/couper).

## Usage

To install Couper via GitHub Releases, use the following configuration:

```json
"features": {
    "ghcr.io/coupergateway/features/couper:1": {}
}
```

## Options

- `version`: Select the version to install (`latest`, `edge`, `1.13`). Default is `latest`.
